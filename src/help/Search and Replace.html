Search and Replace

Selected the Document where you want to search. The current document is chosen by default.

Enter the serach string in the Find:  box.

Enter the replace string in the Replace:  box.

Set the search options. The Use regular expression excludes the other options. For information on how to use Regular Expressions, see Help --> Regular Expressions.

Button's Actions:

Find Next: sequential search of the search string.

Replace: replace the last searchedstring.

Repl & Find: replace the result of the last search and search again.

Replace All: replace all matchs in the document.

Close: close the Find and Replace window.

Help: opens the Help window.
