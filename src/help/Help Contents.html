Introduction

The EMU Editor supports multi-users editing the same document. All shared documents are stored in the EMU Server.

Different types of documents are treated differently: plain text, C programming language and Java. Currently only plain text documents are supported.

A user can edit multiple documents in a single session and each document can be viewed in multiple windows.

The document is seen as a list of atoms. For Plain Text documents an atom is a paragraph.

To see the list of users editing the current document open the users window: Windowâ€“>Show/Hide Users.

Each local editor periodicaly sends to the Server the list of commands executed since the last period. The server periodicaly collects the recent commands received from all editors and broadcast to all clients. The length of the server period is variable and longer when the server becames overloaded.
